# dmfkuii

## Installation

Clone the project

```bash
  git clone https://github.com/hailinzparra/dmfkuii.git
```

Go to the project directory

```bash
  cd dmfkuii
```

Install the project

```bash
  npm run init
```

## Development

Start development watch

```bash
  npm run dev
```

Start Firebase emu

```bash
  npm run serve
```

## Deploy

```bash
  npm run deploy
```
